## Installation

The package can be installed
by adding `markovian` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:markovian, "~> 0.1.1"}]
end
```

The docs can be found at [https://hexdocs.pm/markovian](https://hexdocs.pm/markovian).
